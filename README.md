# Scripts Requerimientos

Repositorio que contiene scripts desarrollados para la automatización y procesamiento de requerimientos en la Diari.

## Contenido

- Scripts en Python para procesamiento de datos
- Automatización de archivos Excel
- Generación de hashes y validaciones
- Consultas SQL asociadas a los procesos

## Estructura

- `/SQL` → consultas SQL utilizadas en los procesos
- Scripts `.py` → herramientas de automatización y transformación de datos
